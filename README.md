# Academic Stress Analytics using PySpark

## About the Project
Academic stress is a growing concern among students and is influenced by factors such as study habits, sleep patterns, attendance, and workload.This project was developed as part of my exploration of Big Data Analytics using PySpark. It analyzes factors affecting students' academic stress by combining distributed data processing, exploratory data analysis, and machine learning.

## What this project does

- Reads and processes student data using **PySpark**
- Cleans and preprocesses the dataset
- Handles missing values
- Performs feature engineering
- Explores stress-related patterns through visualization
- Applies machine learning models to analyze stress levels
- Compares the results of different algorithms

## Machine Learning Models Used

- K-Means Clustering
- Decision Tree Classifier
- Random Forest Classifier

These models were implemented using **PySpark ML (`pyspark.ml`)**.
